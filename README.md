# Lendas do Basquete - NBA Legends

## Tema
Site sobre as maiores lendas da NBA, apresentando jogadores que marcaram época no basquete mundial, como Michael Jordan, LeBron James, Kobe Bryant, Magic Johnson, Larry Bird e outros ícones do esporte.

## Integrantes
- Francisco Santana Almeida - RA: 252259

## Páginas do Projeto
- Home - Página inicial com carrossel, apresentação do tema e cards com as principais lendas
- Galeria - Galeria completa com 9 lendas e filtros por década (anos 80, 90, 2000 e atualidade)
- Contato - Formulário de contato com informações institucionais e mapa de localização

## Tecnologias Utilizadas
- HTML5
- CSS3
- Bootstrap
- JavaScript
- GitHub

## Funcionalidades
- Menu responsivo (Navbar Bootstrap com menu hambúrguer)
- Carrossel Bootstrap com 3 imagens das lendas
- Galeria de imagens com filtros por categoria
- Formulário de contato com validação
- Mapa incorporado (iframe)
- Animações CSS/JS (hover nos cards, transições, botão voltar ao topo)
